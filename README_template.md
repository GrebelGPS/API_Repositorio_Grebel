# Proyecto: Consumo de API

## Descripción
Bueno realmente yo hice un Pokededx (Una Lista D e Pokémon la mejor que veran)

## API utilizada
Use la API de Pokémon Url=[Pokemon api](https://pokeapi.co/api/v2/pokemon)

Esta API da información en tiempo real sobre los Pokémon, incluyendo imágenes, sus tipos, nombre y eso

## Tecnologías utilizadas
- HTML
- CSS (inspirado en los colores de Pokémon) 
- JavaScript (Y el consumo de la API)

## Funcionalidades
Mi pagina es basicamente una mini Pokédex que obtiene datos en tiempo real desde la PokeAPI y los muestra 

## Cómo ejecutar
1. Descargar o clonar el repositorio  
2. Abrir el archivo `index.html` en cualquier navegador web
